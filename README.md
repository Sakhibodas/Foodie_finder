# 🍲 Foodie Finder - Recipe Web Application

Foodie Finder is a full-stack web application that helps users discover recipes across categories such as appetizers, main course, desserts, and beverages. It provides detailed nutritional information and allows users to save their favorite recipes.

---

## 🚀 Features

* 🔍 Search recipes by category
* 🧾 View detailed nutritional values (calories, proteins, carbs, fats)
* ❤️ Save favorite recipes
* 🔐 User authentication (login/logout)
* 👤 User profile with saved recipes
* ⭐ Recipe rating system

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL (XAMPP)

## ⚙️ How to Run the Project

1. Install XAMPP and start **Apache** and **MySQL**

2. Move project folder to:

```id="6y9l1o"
htdocs/
```

3. Import database:

* Open phpMyAdmin
* Create database `foodie_finder`
* Import `final_foodie_finder.sql`

4. Open browser and run:

```id="8v31pp"
http://localhost/foodie_finder
```
