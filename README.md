# Foodie_finder
Developed Foodie Finder, a full-stack web app for discovering recipes with nutritional details. Features include recipe search, user login, and saving favorites. Built using HTML, CSS, JavaScript, PHP, and MySQL (XAMPP), focusing on user-friendly design and efficient data handling.
